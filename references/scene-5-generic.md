# 场景 5：通用自定义

适用信号：未匹配 S1-S4 任何信号，或用户明确自定义需求。

通用八步闭环、执行卡、验证阶梯见 SKILL.md。落盘见 [output-formats.md](output-formats.md) 的通用模板。

## 场景独有要点

Intake 时确认四要素：**目标 / 成功标准 / 约束 / 禁止**。

- 四要素已足够 → 直接进 Inspect。
- 缺的是低风险细节 → 采用保守默认并说明。
- 缺的是目标、成功标准、授权边界，或影响破坏性操作 → 询问，不猜测。

## Prompt 模板

```markdown
用 Loop Mode 执行：
目标：[一句话]
成功标准：[可量化判定]
约束：[必须遵守]
禁止：[绝对不能做]
```
